# CR7 — Cristiano Ronaldo Fan Website 🐐

A stunning 3D aesthetic tribute website for Cristiano Ronaldo built with pure HTML, CSS & JavaScript.

## 🚀 Features
- Animated CR7 hero with gold glow effect & floating SIUUU text
- Custom gold cursor with trail effect
- Scroll progress bar
- 5 auto-rotating image galleries
- Video player with auto-pause (new video = previous stops)
- Full lightbox with prev/next navigation, keyboard & swipe support
- 500 Goals Hall of Fame
- Ronaldo & Georgina love story section
- All CR7 social links & brand links
- Contact section
- Fully responsive

## 📁 Structure

```
CR7_Website/

    ├── images/        ← Profile & misc images

    └── slides/

        ├── slide2/    ← Gallery 2 (Trophies)

        ├── slide5/    ← Gallery 5 (Icons & Moments)
        ├── use/       ← GOAT Collection grid
        └── georgina/  ← Ronaldo & Georgina section
```
